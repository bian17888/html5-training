Video

  • No standard across browsers
  • IE and Safari currently only support MP4
  • MP4 is not free – there are patent/license issues
  • WEBM – the current alternative

Video

  • <video> element
  • Attributes include:
    • src (takes a url)
    • controls
    • autoplay
    • loop
    • poster (takes a url)
    • preload
      • none
      • metadata
      • auto (default)
